Robot localization

The robot will be localized in the workspace using a optical measurement machine, in this case a Nikon K300. It can track LED’s at high resolution in a large workspace.

The camera system will be set to track LED’s in the workspace where it will not be disturbed, since it will define the world coordinate system. This is risky, since if the camera is accidently moved the locations of the coordinate frames would be unrecoverable.

This system will to localize the robot platform after it has been moved. By attaching LED’s to the robot tool, and sampling 10-20 positions of the robot arm (where the cartesian coordinates of the robot flange in relation to the robot base frame is recorded, as well as the location of the LED’s in relation to the camera system).
